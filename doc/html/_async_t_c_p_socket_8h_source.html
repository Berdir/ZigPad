00163 // Once one of the accept or connect methods are called, the AsyncSocket instance is locked in
00164 // and the other accept/connect methods can't be called without disconnecting the socket first.
00165 // If the attempt fails or times out, these methods either return NO or
00166 // call "onSocket:willDisconnectWithError:" and "onSockedDidDisconnect:".
